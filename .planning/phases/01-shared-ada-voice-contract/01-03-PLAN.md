**Step 1 — Generate the HA voice token:**

**Step 2 — Update and SOPS-encrypt the token secret:**
```bash
cd /path/to/home-ops

# Replace placeholder with real token
sed -i "s/REPLACE_WITH_GENERATED_TOKEN_THEN_SOPS_ENCRYPT/$TOKEN/" \
  cluster/apps/openclaw/instance/openclaw-ha-voice-secret.sops.yaml

# Encrypt with SOPS
sops --encrypt \
  --age age16rj2w4scur86mjvs6eutq03cwlpygu3pywcedvl0rkh68vzv0dlqagqenq \
  --encrypted-regex '^(data|stringData)$' \
  --in-place \
  cluster/apps/openclaw/instance/openclaw-ha-voice-secret.sops.yaml

# Verify it's encrypted (should show ENC[AES256_GCM,...] not plaintext)
grep "HA_VOICE_GATEWAY_TOKEN" cluster/apps/openclaw/instance/openclaw-ha-voice-secret.sops.yaml
```

Edit `/config/secrets.yaml` — add:

**Step 4 — Commit, push, wait for Flux:**
